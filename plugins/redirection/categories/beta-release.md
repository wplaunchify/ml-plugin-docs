## Beta releases and old versions

**Source:** [https://redirection.me/support/beta-release/](https://redirection.me/support/beta-release/)

If you want to try a beta release of Redirection, or want to use an older version that you’ve downloaded from WordPress.org, then you can follow these steps:

1. Download the beta (or old version) as a zip file
2. Unzip the file on your computer
3. Using whatever mechanism you use to access your site’s directory contents (i.e. FTP, CPanel etc) navigate to the 
```
wp-content/plugins
```

 directory
4. Rename the existing redirection directory to 
```
redirection.old
```
5. Upload the unzipped redirection folder to 
```
wp-content/plugins
```
6. Confirm that Redirection is still functioning
7. Delete 
```
redirection.old
```

If you follow these steps then no data will be lost.

**Do not delete the old plugin from your plugins page!** This will delete all data associated with the plugin.

Beta versions can be found on the [Redirection Github release page](https://github.com/johngodley/redirection/releases).
