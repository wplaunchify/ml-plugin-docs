					Yes, FooEvents POS works in any country that WooCommerce supports which is practically all countries.

					FooEvents POS is a standalone plugin which means that you don’t need to install any other FooEvents plugin extensions in order to use the standard features of the FooEvents POS app. However, in order to use any of the FooEvents integration features including selling and printing tickets, making bookings and selecting seats, you will need to purchase, install and activate the respective FooEvents plugins that perform this functionality, namely, [FooEvents for WooCommerce](https://www.fooevents.com/products/fooevents-for-woocommerce/) (core), [FooEvents Multi-day](https://www.fooevents.com/products/fooevents-multi-day/), [FooEvents Bookings](https://www.fooevents.com/products/fooevents-bookings/) and [FooEvents Seating](https://www.fooevents.com/products/fooevents-seating/).

							Can I use FooEvents POS without WooCommerce?
					Unfortunately, not at the moment. FooEvents POS is a point of sale solution that is specifically designed and built for WooCommerce and FooEvents.
							How long does the FooEvents POS setup process take?
					FooEvents POS is essentially an extension of your WooCommerce store so you have already done all the hard work! Most customers are up and running within a few minutes.
							Can I run FooEvents POS on any computer?
					Yes! FooEvents POS runs in most modern web browsers including Google Chrome, Safari, Firefox and Microsoft Edge. FooEvents POS can also be used on most tablet devices.

							Does FooEvents POS integrate with Square, Stripe or other payment service providers?
					Yes, FooEvents POS integrates directly with [Square](https://squareup.com/) and [Stripe](https://www.stripe.com/). You can also enter a customer’s card details manually which is perfect for taking orders over the phone or if the hardware is not available in your country.

If you don’t use Square or Stripe, card payments can be manually processed using any third-party card reader of your choice and set to “Card Payment” at checkout. The payment type can also be set to “Cash”, “Direct Bank Transfer”, “Check”, “Cash on Delivery” and “Other Payment Method” or you can create your own custom payment methods in WooCommerce.
							Where can I buy Square and Stripe readers?
					Stripe readers are currently only available to Stripe customers in [select countries](https://support.stripe.com/questions/stripe-terminal-country-and-currency-availability) and you must order readers directly from Stripe through your [Stripe dashboard](https://dashboard.stripe.com/terminal/shop).

You can purchase Square hardware from [Square directly](https://squareup.com/hardware/) or from online and physical retailers in the countries where Square is available such as Amazon.

**Note: You do not need a Stripe or Square reader for manual card payments**
							Does FooEvents POS work with coupons?
					Yes, FooEvents POS integrates with the standard WooCommerce coupons functionality. When completing an order, coupons can be applied to the order which will result in the cart being discounted by the relevant amount.

Please refer to the [Coupons](https://help.fooevents.com/docs/topics/point-of-sale/checkout/#coupons) sectionfor more information.
							How does FooEvents POS handle taxes?
					FooEvents POS uses the built-in WooCommerce Tax Settings when applying taxes to orders. This means that any tax settings and tax rates that have already been configured for your WooCommerce store, will automatically be applied within the FooEvents POS app.

Please refer to the [Tax](https://help.fooevents.com/docs/topics/point-of-sale/tax/) help document for more information.
							Does FooEvents POS support products that are sold in decimal quantities/units of measurement?
					Yes, FooEvents POS has built-in functionality that allows you to sell products using decimal quantities and you can even sell regular products alongside products that use decimal quantities but this first needs to be configured.

Please refer to the[Decimal Quantities](https://help.fooevents.com/docs/topics/point-of-sale/decimal-quantities/) help documentfor more information.
							Can I use FooEvents POS to manage multiple WooCommerce websites?
					Yes, but you will need to purchase an additional licence for each new website/domain.
							Can I customize the FooEvents POS interface colors and logo?
					Yes, you can customize the interface colors and logo that displays in FooEvents POS through the plugin settings.
							Does FooEvents POS store my customer data or transmit it to any third-parties?
					FooEvents POS connects directly to your WooCommerce database using a secure API which fetches your store data (customers, products and orders). None of this information is saved or passed on to any third-party servers, including our own.

WooCommerce handles all the eCommerce functionality and payment processing in exactly the same way as your online store.

FooEvents POS complies with the EU’s General Data Protection Regulation (GDPR) for added peace of mind.
