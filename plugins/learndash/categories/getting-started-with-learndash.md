## Setting Up Your First Course with LearnDash

**Source:** [https://learndash.com/support/kb/core/getting-started-with-learndash/setting-up-your-first-course-with-learndash/](https://learndash.com/support/kb/core/getting-started-with-learndash/setting-up-your-first-course-with-learndash/)

### Step 1: Access the LearnDash Course Builder

1. **Log in to your WordPress dashboard.**
2. **Navigate to LearnDash LMS**: On the left-hand menu, click on 
```
LearnDash LMS
```

 and then select 
```
Courses
```

.

### Step 2: Add a New Course

1. **Click on**
```
Add New
```

: This button is located at the top of the 
```
Courses
```

 page.
2. **Enter Course Title**: Add the title of your course in the provided field.
3. **Add Course Content**:
- In the editor, you can add a course description, images, videos, and any other relevant content. Use the WordPress editor to format and organize your content.

### Step 3: Configure Course Settings

1. **Scroll down to**
```
Course Settings
```

: Here you will find several options to configure your course.
2. **Course Price Type**: Select the pricing model for your course (e.g., Free, Buy Now, Recurring, Closed).
3. **Access Mode**:
- If the course is free, select 
```
Free
```

.
- If it’s a paid course, select 
```
Buy Now
```

 and enter the price.
- For membership-based access, select 
```
Recurring
```

 and set the subscription details.
4. **Course Prerequisites** (Optional): If you want students to complete another course before accessing this one, set the prerequisite course here.
5. **Course Points** (Optional): Assign points to the course if you are using the LearnDash points system.
6. **Course Certificate** (Optional): Attach a certificate that students will receive upon course completion.
7. **Course Materials**: Add any downloadable materials or resources for the course.

### Step 4: Add Lessons and Topics

1. **Scroll up to the**
```
Course Builder
```

: This section is usually located just above the 
```
Course Settings
```

.
2. **Click**
```
Add Lesson
```

: A new panel will open where you can add lessons.
- Enter the title of the lesson and click 
```
Add Lesson
```

.
- Repeat this process to add multiple lessons.
3. **Add Topics to Lessons**:
- Click on a lesson to open it.
- Click 
```
Add Topic
```

, enter the topic title, and click 
```
Add Topic
```

.
- Repeat for additional topics.

### Step 5: Add Quizzes

1. **Scroll to the**
```
Quizzes
```

 section: This is located below the lessons and topics in the 
```
Course Builder
```

.
2. **Click**
```
Add Quiz
```

: Enter the quiz title and click 
```
Add Quiz
```

.
- You can add questions to the quiz by clicking on the quiz title and entering the questions and answers.

### Step 6: Publish Your Course

1. **Review your course content**: Ensure all lessons, topics, quizzes, and settings are configured correctly.
2. **Click**
```
Publish
```

: This button is located on the right-hand side of the screen in the 
```
Publish
```

 box.

### Step 7: Preview and Test Your Course

1. **Preview the Course**: Before making it live, use the 
```
Preview
```

 button to see how the course looks from a student’s perspective.
2. **Test the Course**: Enroll yourself in the course and go through the lessons, topics, and quizzes to ensure everything functions correctly.

### Additional Tips

- **Drip-Feed Content**: If you want to release course content on a schedule, use the 
```
Drip-Feed Content
```

 option in the 
```
Course Settings
```

.
- **Course Forums**: Integrate bbPress to add forums for course discussion.
- **Customize the Course Layout**: Use the LearnDash Focus Mode to enhance the learning experience.

By following these steps, you can create and configure a new course in LearnDash, ready for your students to enroll and start learning.

## Orders

The Order Management system for LearnDash empowers site administrators to manage orders with just a few quick clicks. With improved accessibility, orders are prominently featured in the main navigation menu under LearnDash LMS > Orders, enabling quick and easy access. You can quickly view, search, and organize orders more effectively, enhancing overall workflow and efficiency. You’ll also have access to a mobile view of the orders list, with all the same functionality you enjoy in the desktop view.

![Orders dashboard](https://learndash.com/support/wp-content/uploads/2024/12/orders-01.png)

Orders are included in the main navigation, making them more discoverable. The Orders list has been reorganized to make more important data points easier to find (order ID, Price), optimized row sorting, and added the ability to search by order ID, item, customer email, customer name, and gateway.

## Order details

## FAQ

**Will Orders show transactions outside of LearnDash (i.e. by WooCommerce)?** No, only orders/transactions from LearnDash core will show here. Third-party e-commerce transactions are not supported at this time. **Is Orders compatible with older versions of WordPress?** Yes, it is fully compatible with WordPress version 6.2 or above. Older versions of WordPress can lead to various issues, including the potential malfunction of certain features. **Can I directly manage subscriptions in this UI?** Currently, subscriptions cannot be managed directly in this UI. Anything pertaining to changing end dates, changing subscriptions, etc. is not available at the moment.

## Style Customizer

**Source:** [https://learndash.com/support/kb/core/getting-started-with-learndash/style-customizer/](https://learndash.com/support/kb/core/getting-started-with-learndash/style-customizer/)

The **LearnDash Style Customizer** allows site administrators to adjust the appearance of LearnDash courses, lessons, topics, quizzes, and shared interface elements, **without writing any code**.

The Style Customizer is available in the WordPress Customizer, where changes can be previewed live before being published.

### Accessing the Style Customizer

1. Log in to the WordPress dashboard
2. Navigate to **Appearance > Customize**
3. Select **LearnDash Styles**

The Customizer displays a live preview of the site alongside styling controls specific to LearnDash content.

**Note:** With the introduction of the **Modern Appearance**, some Style Customizer options may not apply as expected. Improvements and refinements are planned for future updates.

## How Styling Works in LearnDash

LearnDash styles can be configured at two levels:

- **Global styles**, which apply to all LearnDash content
- **Page-specific styles**, which apply only to a specific content type (Courses, Lessons, Topics, or Quizzes)

Start by selecting the LearnDash page type to customize from the Customizer panel.

## Using the Style Customizer

Most style controls work consistently across all LearnDash page types.

To update a style:

1. Click **Select Color** (or adjust the available control)
2. Choose the desired value
3. Click **Publish** at the top of the Customizer to save your changes

## Global Styles

Global Styles apply to all LearnDash content unless overridden by page-specific settings.

Available Global Style options include:

- Complete Status Background Color
- Complete Status Text Color
- In Progress Status Background Color
- In Progress Status Text Color
- Course Expand/Collapse Button Border Radius
- Content Header Background Color
- Content Header Text Color
- Next Button Background Color
- Next Button Text Color
- Previous Button Background Color
- Previous Button Text Color

After selecting your preferred values, click **Publish** to apply the changes.

- Current Lesson Text Color

Once satisfied, click **Publish** to save the updates.

## The Design Wizard

After you’ve purchased and built the site, you’ll go through the initial setup wizard to set up how you want your LearnDash site to function. After that, you’ll be taken directly to the **Set Up**tab, where you can see the Design Wizard.

![](https://learndash.com/support/wp-content/uploads/2022/07/Design-wizard.png)

![](https://learndash.com/support/wp-content/uploads/2023/01/learndash-design-wizard.png)

The next step is to choose a font for your website. You can select from different font options provided and click the **Restore**button if you’d like to reset the styles. You can also see these changes in real time on the preview window. Once you are done picking the fonts, click **Next**.

![](https://learndash.com/support/wp-content/uploads/2022/07/design-wizard-_fonts.png)

Finally, you will be presented with a set of color palettes to choose from. This will affect the buttons, links, and even the background color of the page. ![](https://learndash.com/support/wp-content/uploads/2022/07/design-wizard_colors.png)

![](https://learndash.com/support/wp-content/uploads/2022/07/design-wizard-completed.png)

# Coupons

LearnDash coupons are a way for you to easily offer promotions and discounts to your users before they purchase your courses. With LearnDash coupons, you can create coupons for your users. You can even set expiration dates and usage limits for your coupons to make them work the way you like.

**NOTE**: LearnDash Coupons work with the Registration Flow in LD 4.0. You must have the registration option enabled and the registration pages configured to use the coupons for new users. Check out our guide to the [new Registration Flow here.](https://learndash.com/support/docs/guides/the-new-ld-3-6-registration-process/)

LearnDash coupons only apply to courses or groups set to Buy Now. They do not work for recurring courses or groups. If you want to learn more about [Course Access settings, you can check the documentation here](https://learndash.com/support/docs/core/courses/course-access/).

## Creating Your Coupons

To create LearnDash Coupons:

1. Navigate to **LearnDash LMS > Coupons**
2. Click on **Add New**
3. Add a Coupon title
4. Enter the coupon slug – the coupon slug is the unique identifier of the coupon within WordPress. The coupon settings below will still take effect.

From here, you can personalize your coupons to suit your needs.

### Coupon Settings

These settings can help you manage how your coupons are used. We’ll review the configuration options below:

- **Coupon Code** – the actual Coupon Code you can give to your students when they purchase the course.
- **Type** – this is what the coupon will be when applied. You have two options:
- **Flat Rate** – deducts a flat rate from the actual course price
- **Percentage Off** – deducts a percentage off of the actual course price.
- **Amount** – is the actual amount of the coupon, which can be a flat rate or a percentage off the course price.
- **Number of Redemptions** – the number of times the coupon can be redeemed. Enter 0 for unlimited redemptions.
- **Start Date** – the date the coupon can be used.
- **End Date** – indicates the expiry date of the coupon. After this date, the coupon cannot be used.
- **Apply to all Courses** – indicates whether the coupon can be used across all courses. If this is turned off, you have to select the specific courses to which this coupon can be applied.
- **Apply to all Groups** – indicates whether or not the coupon can be used and applied to all groups. If this is turned off, you have to select the specific groups this coupon can be applied to.

Once you’re done updating all the settings, click **Publish**to publish your coupon.

4. Once the registration is successful, you’ll be redirected to the purchase page where you can apply the coupon.
5. If the coupon is valid, the discount will be applied after clicking the **Apply Coupon** button.
6. Click **Take this Course**to proceed to payment. Once the payment is made, you’re successfully registered on the site and enrolled in the course.

### Logged-In/Existing User

Go to the course page of the course you would like to take. Click on the **Login** link located next to the **Take this Course** button.

1. Log in to the site using your existing login credentials.
2. Once you’re logged in, click the **Take this Course** button.
3. You’ll be redirected to a purchase page where you can apply the coupon.
![LearnDash course purchase checkout screen](blob:https://learndash.com/1ccd7ea2-9320-4c9d-9169-5b6bbbe01a4d)
4. Click the **Take this Course** button to proceed to the payment screen.

## The Onboarding Wizard

**Source:** [https://learndash.com/support/kb/core/getting-started-with-learndash/the-onboarding-wizard/](https://learndash.com/support/kb/core/getting-started-with-learndash/the-onboarding-wizard/)

The LearnDash Onboarding Wizard is a new feature introduced in LearnDash version 4.0. It helps students with a fresh LearnDash installation quickly set up their courses, lessons, and payments options. The wizard handles the basic setup process and automatically installs the required plugins to save time.

In just four simple steps, you can go from a new LearnDash install to a fully configured site with add-ons, third-party plugins, and key settings already in place. Once the setup is complete, you only need to add your course content, such as lessons, quizzes, and your site will be ready to use.

## The User Interface

### Step 1: Your Info

The first step in the onboarding experience is validating your LearnDash license. To do this, follow the steps below:

1. Enter the email address you used to purchase LearnDash and
2. Enter the license key that was emailed to you after your purchase.

If you can’t find your license details, click the provided link to log in to your LearnDash account and retrieve the information.

After entering the required details, click Validate to check your license. If the license is valid, you will see a **green confirmation message** that says **“Your license is valid.”**

You can also opt in to the Communication Preferences. This option allows LearnDash to use the email address you entered as the “From” address for certain outgoing emails sent to your customers (such as course purchase confirmations and purchase invoices). These settings can be managed later by going to **LearnDash LMS > Settings > Emails**.

### Step 2: Your Courses

The next step for the Onboarding Wizard is to set up your course structures.

In this section, you can choose whether you want to import a demo course. If you are new to LearnDash, importing a demo course is a helpful way to see how a course is structured and how different elements work together.

Next, indicate how many courses you plan to create. Select either Just One or Multiple. Your selection allows the wizard to automatically configure course listings and related settings based on your needs.

Next, select all that apply to the courses you plan to offer:

- **Simple**– Text-based courses without quizzes or certificates.
- **Certificate**– Video or text-based courses with quizzes and certificates.
- **Timed**– Video or text-based courses with a time limit, including quizzes and certificates
- **Group Courses** – Courses managed by group leaders, ideal for cohorts and corporate training, and classroom use.

If you select Group Courses, you will be asked additional questions such as:

- Are your groups public or private?
- Who will be leading your group courses?

Answering these questions help the wizard automatically configure your group settings.

Selecting the appropriate course types from the above also allows the wizard to determine which plugins and add-ons are best suited for your setup, which you will see in **Step 4**.

Once you are finished, click on Next to continue to the next step.

## LearnDash Overview

**Source:** [https://learndash.com/support/kb/core/getting-started-with-learndash/overview/](https://learndash.com/support/kb/core/getting-started-with-learndash/overview/)

LearnDash is a WordPress learning management system (LMS) trusted by companies, universities, training organizations, and entrepreneurs worldwide for creating and selling their online courses. The structure and delivery of LearnDash courses is based on instructional design principles used in learning programs at Fortune 500 companies:

- The top-level is Course.
- Below that you have Lessons. Lessons are assigned to Courses. Lessons can optionally be grouped into Sections.
- Below Lessons you have Topics. Topics are assigned to Lessons just like Lessons are assigned to Courses.
- Finally, there are Quizzes. Quizzes can be assigned to a Course, Lesson or Topic.

**NOTE**: The terminology can be changed using [custom labels](https://learndash.com/support/docs/core/settings/custom-labels/), but the general structure remains the same. For example, Lessons can be renamed Modules, Quizzes can be renamed to Tests, and so on.

The software itself has a variety of options, and you also have the ability to add functionality using [free and paid add-ons](https://learndash.com/support/docs/add-ons/). For optimal performance and a decreased risk of compatibility issues, we recommend only installing the add-ons that you need.

## Primary Domain vs. Sub-Domain

We are often asked whether it is better to install LearnDash on a sub-domain (ex: 
```
courses.yoursite.com
```

) versus a top-level domain (ex: 
```
yoursite.com
```

).

The short answer is that both options are viable and we have seen successful elearning programs take both approaches.

The benefit to building on a separate domain or sub-domain is that it separates all elearning-related components from those of your main site.

This is useful in case you need to troubleshoot compatibility issues, and it helps to limit any potential conflicts that could occur with the theme/plugins used on your primary domain versus the sub-domain.

However, you can certainly use LearnDash on your primary domain.

If your site is currently focused on training and e-learning, then this probably makes sense. If you have multiple offerings (services, products, e-learning, etc.), then perhaps it is best to separate the e-learning component altogether so as to present a cleaner offering to visitors.

## Shortcodes and Blocks

**Source:** [https://learndash.com/support/kb/core/getting-started-with-learndash/shortcodes-blocks/](https://learndash.com/support/kb/core/getting-started-with-learndash/shortcodes-blocks/)

Shortcodes and blocks provide you with additional ways to insert LearnDash content into your courses, pages, posts and other custom post types. This article will list out every piece of dynamic LearnDash content you can add to your site, and explain how to do it with both a shortcode and a block.

Creating a certificate? Check out our [certificate shortcodes here](https://learndash.com/support/docs/core/certificates/certificate-shortcodes/)

## What’s the Difference?

**Shortcodes**

- The original way to add dynamic content into the WordPress editor
- Can also be used to insert LearnDash content into page builders
- Contain options (also called parameters) to customize the information displayed
- Here’s an example shortcode, with one parameter: 
```
[ld_course_list num="10"]
```

**Blocks**

- Only available when using the new content editor released in WordPress 5.0
- Contain all the same features and options as shortcodes
- Provide an easier interface in which to insert content & customize the options

![List of LearnDash blocks in editor](https://learndash.com/support/wp-content/uploads/2018/09/learndash-blocks-list-in-editor.jpg)

## Which One To Use?

Every shortcode has a corresponding block, and every block has a corresponding shortcode. There are no feature differences between the two. Here are a few questions you can ask yourself to figure out which to use:

- **Using the new WordPress editor (Gutenberg)?** If yes, you can use either. You’ll likely find blocks easier to use, but if you’re comfortable with shortcodes, they are still available to you.
- **Using the Classic Editor plugin?** If yes, than you’ll need to use a shortcode.
- **Using a page builder?** If yes, you’ll need to use a shortcode. Blocks are not yet available in most page builders.
- **Inserting content into a widget/sidebar?** You’ll need to use a shortcode. Blocks are not yet available to use inside of widgets.
