## Group Cohorts

**Source:** [https://learndash.com/support/kb/core/groups/group-cohorts/](https://learndash.com/support/kb/core/groups/group-cohorts/)

LearnDash allows course creators to run cohorts using **Group Start**and **End Dates**. This is helpful when offering the same course to different sets of users at different times, such as seasonal sessions for Spring, Summer, Fall, and Winter.

## How to Enable Group Cohorts

1. Go to LearnDash LMS > Groups
2. Edit any Group
3. Select Settings tab
4. Enter **Start Date** and **End Date** under **Group Access Settings**

![Group Cohorts](https://learndash.com/support/wp-content/uploads/2025/05/Group-Cohorts-800x471.png)

![Can not add Course to a Group Error Message](https://learndash.com/support/wp-content/uploads/2023/08/Screen-Shot-2023-08-16-at-1.57.36-PM.png)

![Can not add Group to a Course Error Message](https://learndash.com/support/wp-content/uploads/2023/08/Screen-Shot-2023-08-16-at-1.58.26-PM.png)

Instead, you can [Clone](https://learndash.com/support/docs/core/settings/cloning/) the course and add it to the new group.

## Group Administration

**Source:** [https://learndash.com/support/kb/core/groups/group-administration/](https://learndash.com/support/kb/core/groups/group-administration/)

Group Administration allows **Group Leaders** to manage learners and track activity within the groups they oversee. When a user is assigned the Group Leader role, they gain access to tools for reviewing assignments and essays, communicating with group members, and exporting group-related reports.

This page explains how Group Leaders access Group Administration and what actions are available to them in the WordPress dashboard.

**Note:** For instructions on assigning the Group Leader role to a user, see the *User Groups* documentation.

## Accessing Group Administration

Group Leaders access Group Administration features after logging in to the WordPress dashboard.

Once signed in, Group Leaders will see a **LearnDash LMS** menu with the following sub-menus:

- **Assignments**
- **Submitted Essays**
- **Group Administration**

These menu items are visible **only** to users who have been assigned the Group Leader role.

## Assignments

When a learner who belongs to one of a Group Leader’s groups submits an assignment, it appears under the **Assignments** menu. From there, Group Leaders can review submissions and approve them if approval is required.

For more information, see the *Manage Assignments* documentation.

## Submitted Essays

Similar to assignments, when a learner completes an essay question within a quiz, the submission appears under the **Submitted Essays** menu. Group Leaders can review and approve essay submissions for learners in the groups they manage.

For more information, see the *Manage Essays* documentation.

## Group Administration Tools

The **Group Administration** menu provides Group Leaders with tools to manage users and export group data. From this menu, Group Leaders can:

- View users in their groups
- Email group users
- Export course and quiz data

### Viewing group users

To view users in managed groups:

1. Navigate to **LearnDash LMS → Group Administration**.
2. Click **List Users**.

This page displays the following information for each user:

- Username
- Name
- Email address
- Actions (Report)

Clicking the **Report** link opens a detailed view showing the user’s total group points. From this screen, Group Leaders can optionally award additional course points.

### Emailing group user

Group Leaders can send an email to all users within a group they manage.

To email group users:

1. Go to **LearnDash LMS → Group Administration**.
2. Click **List Users**.
3. Click the **Email Group** button above the users table.
4. Enter an email subject.
5. Compose your message.
6. Click **Send**.

### Exporting group data

Group Leaders can export group-related data in 
```
.csv
```

 format from **LearnDash LMS > Group Administration**.

The following reports are available:

- **Course Data** (Export Progress):
Displays users’ course progress, current lesson, and course completion status.
- **Quiz Data** (Export Results):
Displays quiz scores, quiz completion dates, and pass/fail status.

## Frequently Asked Questions

### Can Group Leaders be given additional capabilities?

Yes. You can extend the capabilities of the **Group Leader** user role using a plugin such as *User Role Editor*.

### Why aren’t emails sent through Group Administration being delivered?

By default, emails sent through the Group Email feature are delivered using your web host’s mail system. This may not be reliable in all environments. See the email delivery documentation for alternative configuration options.

### Can Group Administration be accessed from the front end of the site?

No. Group Administration is available only within the WordPress admin dashboard.
