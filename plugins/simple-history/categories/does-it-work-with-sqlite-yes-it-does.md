## Does it work with SQLite? (Yes it does!)

**Source:** [https://simple-history.com/support/does-it-work-with-sqlite-yes-it-does/](https://simple-history.com/support/does-it-work-with-sqlite-yes-it-does/)

Simple History does work with the [SQLite Database Integration](https://wordpress.org/plugins/sqlite-database-integration/) plugin from the WordPress team, meaning that you can use Simple History with [SQLite](https://en.wikipedia.org/wiki/SQLite).

It’s a pretty cool way to run WordPress to be honest. No need to set up a MySQL or MariaDB database.
